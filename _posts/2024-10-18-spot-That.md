## Overview

Spot is a web application built with Flask that allows users to monitor their Spotify presence in real-time. This project leverages the Spotify API to enhance user experience by integrating seamless Spotify functionality.

## Features

- Real-time Spotify presence tracking
- Built with Flask and MongoDB for efficient data handling
- User-friendly interface for easy interaction
